### yarn
##### 初始化项目
  ```
  yarn init
  ```
##### 安装指定依赖
 ```
 yarn add [package]

yarn add [package] --dev/-D
 ```
##### 安装包到全局
```
yarn global add [package]
```
##### 安装指定版本的包

```
yarn add [package]@[version]
```
##### 重新下载所有包

```
yarn install --force
```

##### 卸载包
```
yarn remove [package]
```

##### 升级包

```
yarn upgrade
```
